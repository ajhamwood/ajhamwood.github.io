Featured project: Diarie

Diarie is a private diary app with a twist: you use a QR code to unlock access on any device. Featuring private image uploads and a gorgeous interface, Diarie started out as a single-user site—a present to a good friend. It is currently running in demo-only mode, so feel free to create a 1 hour account and try it out for yourself, or check out the code on github.

Proficiencies

My central competencies are in:

  • Client-side web technologies - HTML, CSS and JavaScript
  • NodeJS and JavaScript frameworks such as Express
  • Ruby and Ruby frameworks such as Ruby on Rails, Sinatra and Roda
  • Relational databases such as PostgreSQL, and NoSQL databases such as MongoDB, and
