SMOOTH visualises Stephan Rafler's continuous domain generalisation of the famous Game of Life automaton. By carefully choosing the update rules, one can cause complex structures to emerge - such as gliders, repeaters and still-lifes.

This project features a hand-coded WebAssembly implementation of the Fast Fourier Transform, achieving approximately 96% speed-up over native Javascript.

WORDS is a simple game of word search. Your goal is to find within 3 minutes as many English words as you can using only the 9 letters provided.

The github.io version uses a Web Worker to pre-generate word lists, running blazingly fast while weighing in at under 300 lines of code in a single file.
